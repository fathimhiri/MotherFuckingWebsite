# MotherFuckingWebsite

- Ce repository contient un fichier index.html, ce fichier en plys du html il contient 
aussi du code css dans la balise <style>, et enfin un peu du javascript pour 
dans la balise <script> à la fin du body pour ajouter un peu de vie dans le site.

 - On a utulisé dans la partie JS pour ajouter à ce site une popup JavaScript qui demande à l'utilisateur s'il veut aller sur le site xxxvidsxxx.
et l'activation du mode Psychédélique  lors du click du bouton  "Mode Psychédélique" 
qui entraine: 
1/ Le changement du couleur du backround de la page en magenta.
2/ Le changement la couleur des polices des paragraphes en blue.
3/ Le changement du couleur des polices de tous les headings en vert.
4/ Le changement de la police des paragraphes en courier et la police des titres 
en Comic Sans.
